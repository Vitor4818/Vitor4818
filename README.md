# 👩🏻‍💻 Vitor Gomes Martins

**`Desenvolvedor de Software`**

Sou formado em Análise e Desenvolvimento de Sistemas pela FIAP e atuo como Engenheiro de Software, com foco no desenvolvimento de aplicações escaláveis e na construção de soluções orientadas a boas práticas de engenharia de software.
Possuo experiência no desenvolvimento back-end utilizando Java, Spring Boot e Apache Kafka, além de sólida atuação no ecossistema front-end com React, Next.js, TypeScript e Tailwind CSS. Sou apaixonado por tecnologia, arquitetura de sistemas e aprendizado contínuo, buscando constantemente evoluir minhas habilidades e desenvolver soluções que aliem qualidade, performance e manutenibilidade.
